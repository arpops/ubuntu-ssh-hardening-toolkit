# Ubuntu SSH Hardening Toolkit

Herramienta de auditoría y bastionado para servidores Ubuntu centrada en seguridad SSH.

## Funcionalidades

- Auditoría de configuración SSH
- Detección de configuraciones inseguras
- Comprobación de Fail2ban

## Uso

chmod +x audit/ssh_audit.sh
sudo bash audit/ssh_audit.sh
